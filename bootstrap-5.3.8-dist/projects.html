Projects

Below are selected academic and personal projects that highlight my skills in cybersecurity, programming, and applied problem-solving. Each project not only tested technical knowledge, but also creativity, adaptability, and teamwork.

Course Project – CPRE 4360x

Cybersecurity Steganography Lab (Virtual Machine Deployment)

This lab focused on steganography—the practice of hiding information inside files such as images or audio. I built a custom Ubuntu Linux virtual machine preloaded with analysis tools, allowing users to both embed and detect hidden data. The VM was designed as an interactive learning environment, so students could practice cyber-forensics.

  • Used tools such as OpenStego, StegDetect, and StegHide for encoding and detection.
  • Configured the VM to work across both Linux and Windows systems for accessibility.
  • Packaged the project into a reusable virtual machine for hands-on cybersecurity training.
Course Project – COMS 3090

Work Management Mobile Application

As part of a team project, we developed a workplace management app to improve team communication and scheduling. The app was built with Android Studio (Java/Kotlin) on the front-end and a Spring Boot REST API on the back-end. A PostgreSQL database stored all employee data, time logs, and tasks.

  • Implemented real-time chat, scheduling, and time-tracking features.
  • Tested and validated endpoints with Postman to ensure reliability.
  • Designed UI screens with Android Studio XML layouts, balancing usability and clarity.
Personal Project

Real-Time Face Detection and Recognition

This project explored computer vision by building a Python + OpenCV application that detects and recognizes faces in real-time using a webcam. I extended the project with an alert system that triggers audio notifications when unfamiliar faces are detected, demonstrating a potential application in security systems.

  • Trained the program with sample face datasets for recognition accuracy.
  • Integrated simple SQLite database logic for logging recognized individuals.
  • Explored extending the project toward a web-deployed recognition service.
Personal Project

TryHackMe Cybersecurity Labs

Outside of coursework, I have spent extensive time on TryHackMe, a platform for practicing cybersecurity skills. These labs simulate real-world challenges across networking, web exploitation, privilege escalation, and cryptography.

  • Completed guided labs and capture-the-flag challenges covering multiple security domains.
  • Applied skills such as packet analysis, web penetration testing, and reverse engineering.
  • Built confidence in independent problem-solving and applying theoretical knowledge to hands-on tasks.
